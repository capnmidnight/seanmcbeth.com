PocketDice for PocketPC is a little convenience program for anyone that does anything with dice. I play a number of games that can use any number of dice, so having this program is very handy. No more dice skittering all over everywhere.
-January 17th, 2006
PocketEdit for PocketPC is finished. Unfortunately, the clipboard is not supported in the .NET Compact Framework v1.0, so your clipboard data will not be available to other applications.
I find PocketEdit to be a very handy application for editting text files. Windows Mobile 2003 does not ship with a plain text editor, which can be quite bothersome at times. With PocketEdit, I can edit HTML files while on the go, and quickly view them in Pocket IE.
